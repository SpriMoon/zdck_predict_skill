# 成都中考 5+2 统招线预测器 · 完整参考手册（V1.0 纯净版）

> 本文件是 SKILL.md 的详版支撑材料，含数据来源优先级、系数表、补丁取值表、示例与基线精度。
> 核心机制与硬约束以 SKILL.md 为准；本文件供深度查阅。

---

## 一、适用与触发

- **适用**：成都"5+2"区域（青羊 / 锦江 / 金牛 / 武侯 / 成华 / 高新 / 天府）公办高中**统招**录取最低分预测。
- **触发节奏**：每年 **6 月底**「一分一段表 + 招生计划」发布后方可跑；此前数据不全，不准硬跑。
- **仅覆盖统招**：民办、指标到校、调剂、项目班、郊县班一律拒答（见硬约束 6）。

---

## 二、Phase 0 自主搜寻清单（用户说"跑 202X 届"后执行）

按 5 类数据搜集，来源优先级：

| # | 数据 | 来源 | 备注 |
|---|------|------|------|
| 1 | T-1 年各校统招录取最低分 + 最低位次 R | 成都市教育考试院官网 | 约 40-50 所 |
| 2 | T 年一分一段表（分数→累计人数，全分段） | 成都市教育考试院官网 | 用于 rank↔score 插值 |
| 3 | T 年各校招生计划（**拆 4 渠道**） | 教育局官网 /《招生计划合订本》 | 统招 / 指标到校 / 调剂 / 项目班 |
| 4 | T-1 年各校统招计划 | 同源 | 算 ΔE = T统招 − T-1统招 |
| 5 | T 年新增普通高中名单 + 单校统招计划 | 同源 | 补丁 C 用 |

**可信度层级**：考试院 / 教育局 > "成都教育发布" > 川观 / 成都商报（仅交叉印证）。
**双源冲突**：以考试院为准；差距大标 ⚠️。

**搜不到的处理**：
- 单校缺 → 跳过，输出标 ⚠️「数据缺失」。
- 一分一段只有图 → OCR 或回用户要文字版，**不准估**。
- 招生计划只有总数没拆渠道 → 再搜该校《202X 招生简章》单行本补；补不到用总数估算，输出标 ⚠️「ΔE 为总数估算」。

## 三、Phase 1 数据汇聚（内存三表）

| 表 | 字段 | 说明 |
|----|------|------|
| 表A_history | school \| R_T-1 \| S_T-1 | T-1 各校统招最低位次与分数 |
| 表B_current | score \| cum_rank | T 年一分一段；同时建反向插值 rank→score |
| 表C_plan | school \| regular_T-1 \| regular_T \| ΔE \| quota_T \| transfer_T \| project_T | ΔE 仅算统招渠道 |

**插值规则（线性，输出整数）**：
rank 落在两条记录之间时：
`S = S_a + (R_q − R_a)/(R_b − R_a) × (S_b − S_a)`，结果 `int(round)`。
例：rank=468 落在 rank=402→649 与 rank=474→648 之间
→ 648 + (474−468)/(474−402) × (649−648) = 648.08 → **取整 648**。

## 四、Phase 2 核心机制（系数锁死）

### 4.1 三层主体

```
Step1 等值：表A 取 R → 表B 插值 → S_base
Step2 招生：ΔE = regular_T − regular_T-1，R₁ = R + ΔE
Step3 分流（系数固定，禁止拟合）：
    R₁ ≤ 3000        → D = 0
    3000 < R₁ ≤ 5000 → D = (R₁ − 3000) × 5%
    5000 < R₁ ≤ 8000 → D = 100 + (R₁ − 5000) × 10%
    R₁ > 8000         → D = 400 + (R₁ − 8000) × 12%
    R_final = R₁ + D
Step4 产出：表B 插值 R_final → S_pred（整数分）
```

节点 **3000 / 5000 / 8000** 永久不变；5% / 10% / 12% 永久锁死，禁止回灌。

### 4.2 四补丁（触发式，非叠加式调参）

**补丁A · 首年校（表A 无 T-1 记录）— 锚校法**
`R_X = R_anchor × λ_leader × λ_location`

| λ_leader | 取值 |
|----------|------|
| 四七九本部 | 0.85–0.90 |
| 名校嫡系新校区 | 0.86–0.90 |
| 四七九分校 | 0.92–0.95 |
| 教科院系 | 0.94–0.97 |
| 区属 | 1.0 |

| λ_location | 取值 |
|-----------|------|
| 五城核心 | 0.98 |
| 高新天府 | 1.0 |
| 近郊 | 1.02–1.05 |

**补丁B · 首年校次年（表A 中该校 T-1 是首年）**
`R_source = R_first × (1 + δ_cool)`

| 领办方 | δ |
|--------|-----|
| 七中领办 | +8~12% |
| 石室 / 树德领办 | +5~8% |
| 教科院 | +3~5% |

**补丁C · 同段竞品稀释**
该校 ±50 分区间，T 年新增校统招合计 **≥150** → `R_final += 新增合计 × 0.4`（α 锁死 0.4）。

**补丁D · 异常校弃权**
改制 / 重大舆情 / 统招计划异动 >30% 任一 → 标 🔴「模型失效」，置信 ±5~8 分，**建议人工**。

## 五、Phase 3 双向接口

### 指令①：预测 [校] [年]
四步拆解 + 补丁 + 置信 ±1~2（异常校 ±5~8）。模板见 SKILL.md。

### 指令②：推荐 [分] [年] [区位/调剂偏好可选]
1. 表B 把 S_user 插值得 R_user。
2. 从 R_final 库反查。
3. 梯度窗口：
   - 冲：[R_user−300, R_user−50]
   - 稳：[R_user−50, R_user+200]
   - 保：[R_user+200, R_user+800]
4. 按偏好过滤，每档 3–5 所。

## 六、基线精度（用户知情用）

2026 届 5+2 老校 ~38 所回测：
- ±1 分 ≈ 67%
- ±2 分 ≈ 90%
- 失控由补丁 A/B/C/D 覆盖，不在三层系数内。

---

## 七、硬约束（不可违反）

1. ❌ 禁止用当年真实值回灌调分段系数（5%/10%/12% 永久锁）。
2. ❌ 禁止加第 5、第 6 补丁。
3. ❌ 禁止把 λ / δ / α=0.4 当拟合参数反推。
4. ❌ 分数输出只给整数，不保留小数。
5. ❌ 不准拿"总计划差"当 ΔE，必须拆统招渠道。
6. ❌ 不准预测民办 / 指标 / 调剂 / 项目班 / 郊县班，用户问了拒答。
7. ❌ 跨届不串数据，跑 2028 届时丢弃 2027 重搜。

## 九、自带工具

`scripts/predict.py` 为确定性计算器：把三张表 + 新增/异常元数据写入一份 JSON（见 `assets/input_template.json`），运行即得全量预测线或推荐梯度。系数与补丁在代码内同样锁死，不拟合、不回灌。

```bash
python3 scripts/predict.py --input data.json --run-all
python3 scripts/predict.py --input data.json --recommend 612 --year 2027 --pref 武侯
python3 scripts/predict.py --selftest   # 插值自检
```
